## Overview

A personal portfolio built with Next.js, TypeScript, and Velite for type-safe MDX content. The goal was a fast, accessible site that doubles as a demonstration of my engineering practices.

## Problem

I needed a single place to present my projects, experience, and contact information to recruiters — with enough depth to show how I think about building software.

## Solution

The site uses a single-page layout for quick scanning, dedicated MDX case study pages for deeper project write-ups, and a content layer that validates frontmatter at build time. Dark mode, semantic HTML, and subtle scroll animations keep the experience polished without sacrificing performance.

## Results

- Content updates require only adding an MDX file and pushing to GitHub
- Type-safe project schema catches missing fields at build time
- Deployed on Vercel with automatic previews on every push

## Tech Stack

- Next.js App Router, TypeScript, Tailwind CSS
- Velite for MDX content collections
- Vercel for hosting and CI/CD
