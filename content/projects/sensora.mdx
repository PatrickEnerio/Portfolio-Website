## Overview

Sensora is a full-stack real-time focus tracking dashboard that integrates a camera-based facial analysis pipeline (OpenAI CLIP) with a React frontend, visualizing biometric engagement scores across live and historical sessions.

## Problem

Understanding focus and engagement during study or work sessions is difficult without objective, real-time feedback. Existing tools rarely combine live biometric signals with an intuitive dashboard for both live monitoring and session history.

## Solution

I built a pipeline from camera input through CLIP-based analysis to a microcontroller layer, with a React frontend consuming live focus scores over WebSockets and historical data via REST. A custom SVG line chart normalizes raw sensor data (unix timestamps, 0–1 scores) into display-ready formats.

- **Hardware:** Microcontroller integration for sensor data
