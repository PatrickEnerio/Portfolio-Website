<CaseStudySection title="Context">

Sensora is a full-stack real-time focus tracking dashboard that integrates camera-based facial analysis (OpenAI CLIP) with a React frontend, visualizing biometric engagement scores across live and historical sessions. Built during HackAI in a team of 4.

</CaseStudySection>

<CaseStudySection title="Challenge">

Understanding focus and engagement during study or work sessions is difficult without objective, real-time feedback. Existing tools rarely combine live biometric signals with an intuitive dashboard for both live monitoring and session history — especially within a tight hackathon deadline.

</CaseStudySection>

<CaseStudySection title="Approach">

I built the pipeline from camera input through CLIP-based analysis to a microcontroller layer, with a React frontend consuming live focus scores over WebSockets and historical data via REST. A custom SVG line chart normalizes raw sensor data (unix timestamps, 0–1 scores) into display-ready formats. The UI includes an interactive processing pipeline visualization (Camera → CLIP → Microcontroller) with tooltips and animated charts.

Recognized with Best Beginner Project at HackAI among all first-time hackathon teams.
</Callout>

</CaseStudySection>
