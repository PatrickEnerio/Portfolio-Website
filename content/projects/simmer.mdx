<CaseStudyGrid>

<CaseStudySection title="Context">

Simmer is a full-stack conversational AI agent built for ACM with multi-modal capabilities (chat and voice). It processes 200+ recipe steps and enables hands-free interaction for real customer use cases. I owned all frontend deliverables in a cross-functional Agile team of 6 developers.

</CaseStudySection>

<CaseStudySection title="Challenge">

Home cooks often need step-by-step guidance while their hands are occupied. The product needed to combine voice input, personalized recipe data, and contextual AI responses in a seamless cross-platform mobile experience — all while meeting weekly stakeholder release deadlines.

</CaseStudySection>

<CaseStudySection title="Approach">

I designed and built the React Native UI/UX — layouts, page designs, and navigation routes — while integrating Whisper speech-to-text, text-to-speech, and an OpenAI LLM with custom prompt engineering for real-time Q&A. The API framework connects the AI agent to a recipe database and OAuth 2.0 authentication system. I collaborated closely with backend developers on RESTful API contracts and participated in weekly Agile sprints with stakeholder demos.

</CaseStudySection>

<CaseStudySection title="Impact" colSpan={12}>

- Built end-to-end frontend for cross-platform mobile experience with RESTful API integration across 100+ recipe entries
- Integrated OpenAI LLM with custom prompt engineering for contextual cooking assistance and troubleshooting
- Collaborated in a cross-functional Agile team of 6 developers with weekly releases to stakeholders

</CaseStudySection>
