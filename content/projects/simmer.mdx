## Overview

Simmer is a full-stack conversational AI agent with multi-modal capabilities (chat and voice) built for ACM. It processes 200+ recipe steps and enables hands-free interaction for real customer use cases.

## Problem

Home cooks often need step-by-step guidance while their hands are occupied. A mobile assistant needed to combine voice input, personalized recipe data, and contextual AI responses in a seamless cross-platform experience.

## Solution

I owned all frontend deliverables in React Native—UI/UX layouts, page designs, and navigation routes—while integrating Whisper STT, text-to-speech, and an OpenAI LLM with custom prompt engineering for real-time Q&A. The API framework connects the AI agent to a recipe database and OAuth 2.0 authentication system.

## Results

- Built end-to-end frontend for cross-platform mobile experience with RESTful API integration across 100+ recipe entries
- Integrated OpenAI LLM with custom prompt engineering for contextual cooking assistance and troubleshooting
- Collaborated in a cross-functional Agile team of 6 developers with weekly releases to stakeholders

## Tech Stack

- **Frontend:** React Native
- **Backend:** Python, FastAPI
- **AI:** OpenAI LLM, Whisper STT, text-to-speech
- **Auth:** OAuth 2.0
